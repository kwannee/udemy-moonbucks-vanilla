배운것.

1. 이벤트 위임에 대해서 배웠다. 동적으로 생성되는 element에 이벤트를 달 수 있도록 한다.
2. 이벤트 핸들러 내부에는 인라인으로 작성하지 말고 따로 함수를 분리 시켜서 작성하자. 나중에 어떤 일을 하는 코드인지 알기 편하다.
3. this의 사용에 대해서 좀 더 명확하게 배웠다.
4. 아름다운 코드 분리를 봤다. 명확한 기준을 가지고 코드를 분리 해 중복되는 코드를 최소화한다.

4번이 제일 좋았다. $ 함수를 만들어 긴 selector를 짧게 줄일 수 있었고, request와 HTTP_METHOD 함수를 따로 만들어 fetch 코드를 최소화 한점도 아름다웠다.
